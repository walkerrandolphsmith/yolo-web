Overview

Yolo is an Android application to help parents enforce no texting while driving. Parents can use Yolo to become aware of when their children are in moving vehicles and remotely lock the device in two steps: Register an account and add children's devices. Locked devices have the ability to make emergency, 911 calls and up to 4 personal contacts can be reached!

Installation

β testers can download Yolo, by selecting the link when viewing this website on an Android device.

Sign Up

Accounts are for parents to monitor their children's devices.

  1. Install application on parent's phone.
  2. Open application and select Sign In button.
  3. Select Sign Up button.
  4. Navigate to website.
  5. Select Parents button.
  6. Select Sign Up button.
  7. Fill out form and select Register button.
Parents registering from website must pair their own physical device in order to receive push notifications from their children.

Sign In

Parent's sign in to gain access to updating their settings and adding children's devices to their account.

  1. Open Application
  2. Select Sign In button.
  3. Navigate to website
  4. Select Parents button.
  5. Fill out form using username and password.
  6. Select Sign In button.

Add Child's Device

Adding a child's device to a parent's account allows the parent to manage and monitor the child's device.

  1. Sign In, on the child's device.
  2. Select the phone icon on the menu.
  3. Add a name associated with the child's device.
  4. Select Add Device button.
  5. Sign Out using the back arrow on the menu!
Children must be added to the parent's account using the app on the child's physical device. The website can only help parents monitor children they have already added.

Manage Children

Managing children allows the parent to edit or delete children's devices associated with their account.

  1. Sign In, on your device.
  2. Swipe left on child.
  3. Select the trash can icon to remove child's device from account.
  4. Select clipboard icon to edit information about the child's device.

Monitor Children

Monitoring children's devices allows the parent to receive notifications from the child's device. In addition parents can remotely lock the child's device with a password and optional expiration.

  1. Sign In, on parent's device.
  2. Swipe left on child.
  3. Select the lock icon.
  4. Fill out form with password and expiration.
  5. Select Lock button.

Settings

Allows parent to update notification preferences, message frequency, account settings.

Parents registered via the website must pair their device to their account in the settings menu in order to receive push notifications from their children.
  1. Sign In
  2. Select the settings icon on the menu.
  3. Select the drop down arrow next to your username.
  4. Select settings.

Notification Preferences

Parents can be notified when their children are in moving vehicles by zero or more ways including email, text message, and push notifications.

Reminder Frequency

Allows the parent to determine the minimum amount of time that must elapse since the last message to receive another message.

Account Settings

Allows the parent to edit the password, phone number, and email address associated with their account.

Pair Device

Allows the parent associate a physical device with their account in order to receive push notifications from children's devices.
