Installation

Individual Use

The application can be used for personal use for those who wish to put texting while driving limits on themselves.

Parental Use

The application can be used by parents to enforce no texting while driving.

Sign Up

Accounts are for parents to monitor their children's devices.

  1. Install application on phone.
  2. Open application and select login.
  3. Select Sign Up button.
  4. Fill out form and select register button.

Sign In

Sign into your account to add and remove devices associated with children, change account settings, and monitor childrens' devices.

  1. Open Applications
  2. Select Sign In button
  3. Fill out form using username and password.
  4. Select Sign In button.

Add Child

Adding children allows the parent to receive notifications from the child.

  1. Sign In, on the child's device.
  2. Select the phone icon on the menu.
  3. Add a name associated with the child's device
  4. Select add device button
  5. Sign Out using the back arrow on the menu!

Monitor Children

Monitoring children allows the parent to edit or delete children's devices associated with their account. In addition the parent can remotely lock a child device with a password and expiration ranging form 5 minutes to no expiration.

  1. Sign In
  2. Swipe left on child
  3. Select the lock icon
  4. Fill out form with password and expiration.
  5. Select lock button.

Settings

Notification Prefrences

Allows parents to determine which type of notifications they perfer to receive including email, text message, and push notifications.

Reminder Frequency

Allows the parent to determine the minimum amount of time that must elapse since the last message to receive another message.

Account Settings

Allows the parent to edit the password, phone number, and email associated with their account.

Pair Device

Allows the parent associate a physical device whith their account in order to receive push notifications from children's devices.
